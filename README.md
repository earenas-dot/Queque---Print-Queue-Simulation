# Simulación de Cola de Impresión

## Descripción

Proyecto que simula una cola de impresión utilizando la estructura de datos Queue (FIFO).

El sistema genera trabajos de impresión, los almacena en una cola y una impresora los procesa en orden de llegada.

## Tecnologías

- Python
- CustomTkinter

## Características

- Queue personalizada
- Simulación temporal
- Procesamiento FIFO
- Métricas
- Interfaz gráfica
- Validaciones
- Pruebas básicas
